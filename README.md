This is a fun Python program that generates random **positive and funny sentences**. Each time you press **Enter**, you get a new sentence to brighten your day. You can stop the program by typing `q`.

- Keeps running until you decide to quit
- Easy to customize with your own subjects, verbs, and objects

## How to Use

1. Make sure you have Python installed (Python 3.x recommended).
2. Download the `sentence_generator.py` file.
3. Open a terminal or command prompt in the folder where the file is saved.
4. Run the program:

```bash
